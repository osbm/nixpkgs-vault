# gnomeExtensions.disable-hover-on-app-window-switcher-popups-for-45

## üìù Description

Prevents the mouse cursor interfering with either the App Switcher or the Window Switcher popups.
Note: It doesn't prevent mouse clicks.

These are the switcher popups that are invoked from keyboard shortcuts, e.g. Alt+Tab or Super+Tab.
