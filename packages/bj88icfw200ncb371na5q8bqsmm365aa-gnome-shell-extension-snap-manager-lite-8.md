# gnomeExtensions.snap-manager-lite

## 📝 Description

Popup menu in the top bar to easily manage usual snap tasks (list, changes, refresh, remove, install...).

 All the results of actions are displayed in GNOME Terminal, so you can exactly know what you are doing and what is processing. Very light extension, no background process, no periodic background task.

 You can disable snaps auto updates for some time and consequently manually manage them.

 This is not an official Ubuntu/Canonical extension.
