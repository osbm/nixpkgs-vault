# gnomeExtensions.desktop-clock

## üìù Description

Add widgets to the desktop.

Widgets include:
- Digital Clock
- Analog Clock
- Weather Widget
- Static Text Widget
- Run terminal commands on a user defined interval with output shown as a text widget

Widgets can be moved to your desired location by dragging them with the mouse.
Note: Desktop Icons extension will interfere with this feature. Temporarily turn off Desktop Icons to move the widgets.
