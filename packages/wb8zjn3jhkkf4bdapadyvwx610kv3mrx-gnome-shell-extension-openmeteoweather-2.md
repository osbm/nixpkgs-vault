# gnomeExtensions.openmeteoweather

## 📝 Description

Display the current weather in the panel. The indicator position can be adjusted in preferences.

You need an active internet connection to retrieve and display weather information.

This extension is using `open-meteo.com` services.

This is a fork of the Weather or Not extension.
