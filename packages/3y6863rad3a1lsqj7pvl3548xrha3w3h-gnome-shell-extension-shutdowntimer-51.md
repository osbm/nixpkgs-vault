# gnomeExtensions.shutdowntimer

## 📝 Description

Shutdown/reboot/suspend the device after a specific time or wake with a rtc alarm.

The screen-saver will not interrupt the timer. A privileged control script may be installed to control shutdown and rtcwake as user.
