# gnomeExtensions.logo-menu

## üìù Description

Logo Menu - Menu similar to Apple's macOS menu for the GNOME Desktop
This extension gives a simple menu along with the ability to get the icon of your distro on top left part of the panel for a great look.
The Icon can be customised through settings, it has both Linux and BSD logos.
For more screenshots, visit GitHub.

The 'Activities' indicator is hidden by default but can be enabled through settings.

The default Terminal and Software centre can also be changed.

 Force Quit works on Wayland and Xorg
