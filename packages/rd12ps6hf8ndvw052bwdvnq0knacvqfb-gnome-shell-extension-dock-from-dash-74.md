# gnomeExtensions.dock-from-dash

## üìù Description

Dock using native GNOME Shell Dash.

 Very light extension.

 Hover the bottom of your screen and GNOME Shell dash will appear without overview and will hide when you leave the dash. Native GNOME Shell click behavior is modified: minimize if one window is open, overview if many windows are open. Scroll on the dock to change workspace.
