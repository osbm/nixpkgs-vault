# gnomeExtensions.screen-word-translate

## 📝 Description

Translate word on the screen.
Default web address is translate.google.com, you can add the web address for your own language. Also you can contribute your web address to my github repo.
Use hotkey Ctrl+Alt+j to toggle the function.
Use hotkey Ctrl+Alt+o to show popup window
