# gnomeExtensions.serenity-desktop

## üìù Description

A Per-Monitor-Workspace window manager designed for productive use. It offers two main features:

  - Per-monitor workspaces: Compatible with both static and dynamic workspaces.

  - Application/Window shortcuts: Assign shortcuts to windows and bring the workspace containing the window into view, regardless of its location. Shortcuts are assigned using Super-Shift-Number and can be referred to with Super-Number. Note that this may conflict with default shortcuts, which must be resolved after installation.

Visit the homepage for usage details, backports, and additional post-installation instructions.
